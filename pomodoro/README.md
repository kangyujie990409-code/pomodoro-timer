# 桌面番茄钟

一个用 Python + Tkinter 写的轻量番茄钟，无第三方依赖，双击即用。

## 运行

任选其一：

- 双击 `启动番茄钟.bat`（推荐，无控制台窗口）
- 双击 `pomodoro.pyw`（若 `.pyw` 已关联到 `pythonw.exe`）
- 命令行：`pythonw pomodoro.pyw`

要求：已安装 Python 3.8+（含 Tkinter，标准库自带）。本机 Python 3.13 已就绪。

## 功能

- 三种阶段：**专注 / 短休息 / 长休息**，按设定的"几个番茄后长休息"自动轮换
- **环形进度** + 大字倒计时
- **开始 / 暂停 / 重置 / 跳过** 四个核心操作
- 阶段结束 **系统提示音**；未开启自动连播时会弹窗提醒
- **窗口置顶** 复选框
- **设置面板**：自定义四个时长参数、是否自动开始下一阶段
- **每日完成数** 自动按本机日期归零；配置写入 `~/.pomodoro_config.json`

## 默认参数

| 项 | 默认值 |
| --- | ---- |
| 专注 | 25 分钟 |
| 短休息 | 5 分钟 |
| 长休息 | 15 分钟 |
| 长休息间隔 | 每 4 个番茄 |
| 自动开始下一阶段 | 是 |
| 窗口置顶 | 否 |

## 创建桌面快捷方式（可选）

1. 右键 `启动番茄钟.bat` → 发送到 → 桌面快捷方式
2. 在快捷方式属性里可改图标

## 配置文件位置

`%USERPROFILE%\.pomodoro_config.json`

删除即可恢复默认。
